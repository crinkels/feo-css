This is how You implement it. Note: the additional div is required to ensure the table is sidescrollable and does not break layouts on small screens. 

<div>
  <table>
    <thead>
      <tr>
        <th></th><th>...</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td></td><td>...</td>
      </tr>
    </tbody>
  </table>
</div>

There are several custom properties available that can be overwritten to control the looks of the table.

Custom property Description
--table-radius Sets the radius of the corners
--table-color Sets the color of the borders and header
